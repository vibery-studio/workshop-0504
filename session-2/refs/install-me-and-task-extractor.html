Install me + task-extractor

Two starter skills, four ways to install. Pick your surface and follow the steps. Most people: 4 minutes.

What you're installing: two markdown-only skills that load who you are (me) and what you do (task-extractor) into Claude. Both live as folders with SKILL.md at the root. Source repo: github.com/bnqtoan/claudekit.

01Claude.ai web (most attendees)

For everyone on Claude Pro / Max / Team — this is the path.

Prerequisite: enable Code execution

  1. Click your profile (top-left) → Settings.
  2. Open Capabilities.
  3. Toggle "Code execution and file creation" on.

Without this, the Skills menu is hidden — most common reason "I don't see Skills."

Install the two skills

  1. Download both ZIPs from the workshop release: me.zip and task-extractor.zip (link Tony shares in workshop chat — also from the GitHub repo Releases page).
  2. In Claude.ai sidebar → Customize → Skills.
  3. Click + → Create skill → Upload a skill.
  4. Select me.zip. Wait ~5 sec for processing.
  5. Toggle the skill on.
  6. Repeat for task-extractor.zip.
  7. Open a new chat and type /me — the skill should fire.
Gotcha: The ZIP must contain a folder with SKILL.md at its root, not nested one extra level. If upload fails or skill doesn't trigger, re-zip so the structure is me/SKILL.md (not me/me/SKILL.md). This is the #1 install failure.

02Claude Desktop (macOS / Windows)

Claude Desktop is a wrapper around the web app — same install flow.

  1. Open Claude Desktop.
  2. Settings → Capabilities → enable "Code execution and file creation".
  3. Settings → Customize → Skills → + Create skill → Upload a skill.
  4. Upload me.zip, then task-extractor.zip. Toggle each on.

03Claude Code CLI (terminal users)

Skills live as folders in your home directory. Three install paths.

Path A — Single copy-paste (recommended)

git clone --depth 1 https://github.com/bnqtoan/claudekit.git /tmp/ck && \
  cp -r /tmp/ck/skills/me ~/.claude/skills/me && \
  cp -r /tmp/ck/skills/task-extractor ~/.claude/skills/task-extractor && \
  rm -rf /tmp/ck

Restart your Claude Code session — skills auto-discovered.

Path B — Plugin marketplace (cleanest one-liner)

/plugin marketplace add bnqtoan/claudekit
/plugin install me
/plugin install task-extractor

Run inside a Claude Code session. Marketplace flow handles updates too.

Path C — Manual ZIP unzip

unzip me.zip -d ~/.claude/skills/
unzip task-extractor.zip -d ~/.claude/skills/
Personal vs project: ~/.claude/skills/ = available everywhere (personal). .claude/skills/ in a project root = available only in that project (handy for team-shared skills via Git).

04Cowork

Same ZIP flow as web, with a richer marketplace UI.

  1. Open Cowork → Settings → Customize → Skills.
  2. Either Browse (org marketplace if your admin published the skills) or Upload (custom ZIP).
  3. Upload me.zip, then task-extractor.zip. Toggle on.
  4. Bonus: in any chat, run /skill-creator to spin up new skills without leaving Cowork.

05Verify it worked

  1. Start a new chat.
  2. Type "build my me.md" or /me.
  3. The skill should ask you 7 questions (role, audience, vocab, etc.) and write a me.md.
  4. Same for /task-extractor — should ask about ONE task and write task.md.

If neither fires: skill is uploaded but disabled. Toggle on. If still nothing: ZIP structure wrong (see Gotcha above).

06Where the files end up

  • claude.ai / Desktop / Cowork: stored server-side, attached to your account. Output me.md appears in the chat — copy it into your Claude Project as a knowledge file so it loads automatically across every chat in that project.
  • Claude Code CLI: output written to ~/.claude/me.md directly. Loaded automatically.
